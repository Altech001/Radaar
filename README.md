## ğŸ’¾Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Altech001/Radaar.git
   cd Radaar/
   ```

2. **Install dependencies:**
   Using **Bun**:
   ```bash
   bun install
   ```

   Using **npm**:
   ```bash
   npm install
   ```

3. **Start the development server:**
   Using **Bun**:
   ```bash
   bun dev
   ```
   Using **npm**:
   ```bash
   npm run dev
   ```

4. **Open the project in your browser:**
   Navigate to `http://localhost:8080`.
